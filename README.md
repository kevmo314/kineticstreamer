# Kinetic Streamer

Kinetic Streamer is a mobile app that turns your phone into an RTSP camera.

## Features

- [x] Run an RTSP server on your phone
- [x] H.264, H.265, VP8, VP9, and AV1 video codecs
- [x] Circular buffer recording for video on disk
- [x] Opus audio codecs
- [ ] Client support
  - [x] SRT
  - [x] WHIP
  - [ ] RTSP
  - [ ] RIST
  - [ ] RTMP
- [ ] Server support
  - [x] RTSP
  - [x] WHEP
  - [x] HLS
 
## Roadmap
